# Датасет

Данные для обучения сгруппированы по папкам для каждого мема отдельно.

```
data/
├── 1/          # Мем №1
├── 2/          # Мем №2
├── ...
└── 8/          # Мем №8
```

Для каждого мема было отснято по 5 видео в разнообразных условиях, для усиления обучения. В рамках 2-10 секнуд видео, человек принимает характерную для мема позу и выражение лица.
